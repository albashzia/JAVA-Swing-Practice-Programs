# JAVA Swing Practice Programs

This repository contains a collection of small Java programs created while learning **Java Swing**, the GUI toolkit included in the Java Standard Library. The purpose of the repository is to practice and explore how graphical user interfaces are built using Swing components and layouts.

The programs focus on experimenting with different UI elements and understanding how they interact with event handling, layout managers, and container components.

---

## Overview

Java Swing provides a set of tools for building desktop graphical applications in Java. While learning these concepts, individual programs were written to explore how specific components behave and how they can be combined to build functional interfaces.

This repository organizes those practice programs into small, focused examples that demonstrate the usage of common Swing components and layout managers.

Each program typically focuses on a single concept or component to help isolate and understand its behavior.

---

## Topics Covered

The repository includes practice examples related to the following Swing components and concepts.

### Containers

These components serve as the foundation for building GUI applications.

- **JFrame** – Top-level application window  
- **JPanel** – Generic lightweight container used to organize components  
- **JLayeredPane** – Container that allows components to overlap in layers  

### Basic Components

- **JLabel** – Displaying text or images  
- **JButton** – Interactive button elements  
- **JTextField** – Single-line text input field  
- **JCheckBox** – Toggleable checkbox inputs  
- **JRadioButton** – Mutually exclusive selection options  
- **JComboBox** – Dropdown selection menus  
- **JSlider** – Adjustable value sliders  
- **JProgressBar** – Progress indicators  

### Dialogs and Utility Components

- **JOptionPane** – Simple dialog boxes for user interaction  
- **JColorChooser** – GUI component for selecting colors  
- **File Chooser / File Operations** – Basic interaction with the file system  

### Event Handling

Examples demonstrating how Swing components respond to user interactions.

- **ActionListener**  
- **KeyListener**  
- **MouseListener**

These programs help demonstrate how events are triggered and handled inside Swing applications.

### Layout Managers

Swing uses layout managers to control how components are arranged within containers. Practice examples include:

- **FlowLayout**  
- **BorderLayout**  
- **GridLayout**

These examples show how different layout strategies affect component positioning and resizing.

---

## Purpose of the Repository

This repository serves as a personal practice space for learning Swing concepts through hands-on coding. Writing small, isolated programs helps with understanding how components behave and how they can be combined to form larger GUI applications.

The examples are primarily educational and focus on experimentation with Swing features rather than building full applications.
